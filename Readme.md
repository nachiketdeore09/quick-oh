# For Initialising this project:

This are the Dependencies

```
npm install express mongoose cors dotenv multer cloudinary bcryptjs jsonwebtoken cookie-parser mongoose-aggregate-paginate-v2

```

This are Dev-Dependencies

```
npm install --save-dev nodemon

```
